# React Nanodegree project 3
> React Native

Built with:
- [React Native](https://facebook.github.io/react-native/)
- [Redux](https://redux.js.org/)
- [Expo](https://expo.io/)

The application was built for iOS and has been tested on iPhone 5

## Insructions

Run
```bash
npm install
```
or
```bash
yarn install
```

Start the app:
```bash
npm start
```
or
```bash
yarn start
```
